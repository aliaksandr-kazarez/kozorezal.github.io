What is cross-platform development?

Ability of a programming language (such as Java) that enables programmers to develop software for several competing platforms by writing a program only once. Cross-platform software can run on most or all systems with little or no modification
— businessdictionary.com

Write once, run anywhere

— Sun Microsystems

Write Once, Debug Everywhere

— Wong, William

Options

  • Binaries

    C, C++ Compiled for each platform individually.

  • Bytecode

    Java, C# Compiled in bytecode, then JIT for each platform.

  • Scripts

    Python, JS interpreted on each platform.

Approaches

  • Single sources

    All sources for every platform in one repo

  • Different sources

    Each platform has it's own repo

  • Mixed sources

    Platform agnostic code in one repo. Platform specific in own repos
